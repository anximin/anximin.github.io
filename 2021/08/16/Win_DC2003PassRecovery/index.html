域控【2003】密码重置

如果Windows Server 2003域控服务器的密码忘记了,可以在这里下载Mcse_PassRecovery工具。
Windows Server 2003没有提供恢复域控制器密码的工具，此工具来自MSCE技术论坛，而此论坛目前好像已经不存在了。如果你有它的消息欢迎提供线索。

忘记域控制器密码的解决办法：

1.将下载的文件解压。并将文件夹拷贝到目标域控C盘根目录下。
2.重新启动电脑按F8启动Windows 2003到目录服务恢复模式（Directory Service Restore Mode）。
注：有关目录还原模式密码破解可以使用一些PE工具盘中的清除密码工具修改。
3.执行setup.EXE文件,执行过程会自动在域控添加一个mcse用户并添加管理权限，执行完成后将自动重启。
4.进入正常模式后，使用用户名：mcse 登录，密码为：P@ssw0rd。
5.登录后修改administrator用户权限以及用户密码。
6.修改后务必手工删除mcse用户。

注意：此操作属于第三方工具的引用。解压时杀软就会报毒，我也没有经过长期测试。
因此，不能对其的安全性以及对系统的稳定性的影响提供任何形式的评估。
